## Rest Api란?

- 정보를 주고 받는 형식
- 자원을 이름과 상태로 구분하여 전달
- 요청의 의도와 목적을 쉽게 파악할 수 있음

## Rest api Method 종류

- get : 조회
- post : 정보 추가
- delete : 삭제
- put : 수정 - 전체
- patch : 수정 - 부분
